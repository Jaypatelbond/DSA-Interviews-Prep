# DSA-Interview-Prep (Kotlin)

A curated collection of Data Structures & Algorithms solutions focused on Kotlin.

This repository contains Kotlin implementations of common interview problems organized under `src/kt/`.

Contents
- `src/kt/` — Kotlin solutions organized by topic (binary tree, recursion, sorting, etc.)

Goals
- Keep the repository Kotlin-first so contributors can submit idiomatic Kotlin solutions.
- Provide simple guidelines for adding new problems and tests.

How to use
- Open the project in IntelliJ IDEA (recommended) or use the Kotlin compiler (`kotlinc`) to compile and run individual files.

Examples
- `src/kt/binarytree/` — tree utilities and sample problems
